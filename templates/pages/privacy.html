Privacy Policy

Last updated: March 2026

Your files stay on your device

imgpact processes all images and files entirely in your browser using WebAssembly. No image data, file names, or file contents are ever transmitted to our servers. We have no technical ability to access your files.

Tool usage statistics

When you use a tool, imgpact sends a lightweight anonymous event to our server containing:

  • The name of the tool used (e.g. crop, gif-maker)
  • A random session ID generated by your browser (imgpact-sid, stored in localStorage)
  • A timestamp

This data contains no file data, no IP address, and no personally identifiable information. It is used solely to count tool uses and estimate the number of unique users. The aggregated results are publicly visible on the Statistics page. This tracking is always active, it does not require analytics consent because no personal data is collected.

Analytics (opt-in)

With your consent, we use Google Analytics (GA4) and Microsoft Clarity to collect anonymous, aggregated usage statistics, such as which tools are most used, how long visits last, and what device types are common. This helps us improve imgpact and prioritise new features.

This data contains no personally identifiable information and no file data. You are asked for consent on your first visit, and you can change your preferences at any time using the Cookies button at the bottom-left of the page.

If you decline analytics, Google Analytics still loads in a privacy-preserving anonymous mode (no cookies, anonymised IP). Microsoft Clarity is not loaded at all.

Local storage

We store the following data locally in your browser:

  • imgpact-theme, your light/dark mode preference (never sent to our servers)
  • imgpact-lang, your language preference (cookie, 365 days, never sent beyond page requests)
  • imgpact_consent, your cookie consent choices (365 days, never sent to our servers)
  • imgpact-sid, a random anonymous session ID used for tool usage counting (sent to our server only when a tool is used, alongside the tool name)

Third-party services

We use the following external services which may log your IP address as part of standard HTTP request handling:

  • Cloudflare, CDN and DDoS protection
  • Google Fonts, typography
  • jsDelivr, icon library

Data retention

Tool usage events (tool name + anonymous session ID + timestamp) are stored indefinitely on our server and contribute to the public statistics. Analytics data is retained for 14 months in Google Analytics (default setting) and 90 days in Microsoft Clarity.

Your rights (GDPR)

If you are located in the EU/EEA, you have the right to access, correct, or delete any data held about you, and to withdraw consent at any time. The only data we store on our servers is an anonymous random session ID paired with a tool name, with no way to link it to a real person. You can withdraw analytics consent at any time via the Cookies button.

Contact

For privacy-related questions, please contact us via the Help page.
