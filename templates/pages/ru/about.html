Об imgpact

imgpact. набор бесплатных браузерных инструментов для работы с изображениями и GIF. Без аккаунтов, без загрузки файлов, без ограничений.

Как это работает

Каждый инструмент на imgpact работает полностью в вашем браузере с использованием WebAssembly (WASM). Ваши файлы никогда не покидают устройство. никаких серверов, обрабатывающих изображения, никакого облачного хранилища и никакого сбора данных, связанных с вашими файлами.

Почему imgpact?

  • Без ограничений по размеру файла. обрабатывайте файлы настолько большие, насколько позволяет память браузера.
  • Без ограничений по количеству файлов. пакетно конвертируйте или оптимизируйте столько файлов, сколько нужно.
  • 100% конфиденциально. ваши изображения остаются на вашем устройстве.
  • Мгновенно. никакого ожидания загрузки, никаких очередей.
  • Бесплатно. все инструменты бесплатны навсегда.

Инструменты

Инструменты для изображений

  • Сжатие и оптимизация. уменьшение размера файла при сохранении качества (JPEG, PNG, WebP)
  • Изменение размера. изменение размера по пикселям или процентам
  • Обрезка. произвольная или с фиксированным соотношением сторон
  • Добавить текст. наложение текста с настройкой шрифта и цвета
  • Эффекты. оттенки серого, размытие, яркость, контраст, сепия и другое
  • Трансформация. поворот, отражение и конвертация формата

Инструменты для GIF и видео

  • Создание GIF. создание анимированного GIF из набора изображений
  • Редактор GIF. изменение порядка кадров, настройка тайминга, обрезка, поворот, отражение, реверс
  • Анализатор GIF. просмотр количества кадров, задержек, размеров и палитры
  • Разбивка GIF. извлечение отдельных кадров из анимированного GIF
  • GIF в MP4. конвертация анимированного GIF в видео MP4
  • GIF в WebM. конвертация анимированного GIF в видео WebM
  • GIF в MOV. конвертация анимированного GIF в QuickTime MOV
  • Видео в GIF. конвертация видеоклипа (MP4, WebM, MOV) в анимированный GIF

Статистика использования

imgpact регистрирует анонимные события использования инструментов, чтобы понять, какие из них наиболее полезны, и направлять дальнейшее развитие. Данные о файлах и личная информация не собираются. только название использованного инструмента и случайный идентификатор сессии, сгенерированный браузером. Все агрегированные статистики публично доступны на странице Статистики.

Технологии

imgpact построен на Rust, скомпилированном в WebAssembly для обработки изображений, и Axum для раздачи страниц. Движок изображений использует крейт image для конвертации форматов и крейт gif для работы с GIF. Для обработки видео используется FFmpeg.wasm. Анонимные события использования хранятся в лёгкой базе данных SQLite на сервере.
