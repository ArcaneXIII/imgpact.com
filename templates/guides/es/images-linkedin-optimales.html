Imágenes Óptimas para LinkedIn 2026

⏱ 5 {{ t.guide_min_read }} 📅 {{ t.guide_updated }}

LinkedIn es la principal red social profesional con más de 1.000 millones de miembros. Las publicaciones con contenido visual superan consistentemente a las publicaciones solo de texto en alcance e interacción. Tener las dimensiones correctas de tus imágenes garantiza que tu contenido luzca pulido y profesional, y evita las imágenes pixeladas o recortadas que dañan tu imagen de marca.

Tu foto de perfil de LinkedIn es una de las primeras cosas que la gente ve. Sube al menos 400×400 px y asegúrate de que tu cara o logo esté centrado, LinkedIn recorta en forma circular. Usa una foto de perfil profesional con un fondo limpio y sin elementos que distraigan.

Formato con mejor rendimiento: Según los propios datos de LinkedIn, las imágenes cuadradas 1:1 (1080×1080) generan más interacción que las imágenes horizontales en el feed. Las imágenes verticales ocupan aún más espacio vertical pero pueden recortarse en móvil. El formato cuadrado es la opción más segura para máxima visibilidad.

Buenas prácticas para imágenes en LinkedIn

  • Usa PNG para gráficos con texto: la compresión JPEG de LinkedIn desenfoca el texto. Usa PNG para diapositivas, citas e infografías
  • Sube JPEG de alta calidad para fotos: calidad 85–90, espacio de color sRGB
  • Evita texto pequeño: las imágenes del feed se muestran a unos 504×262 px en escritorio, asegúrate de que el texto sea lo suficientemente grande para leerse a ese tamaño
  • Añade subtítulos/texto a las imágenes: LinkedIn no añade subtítulos automáticamente; agregar superposiciones de texto aumenta la accesibilidad y la interacción
  • Consistencia de marca: usa tus colores, fuentes y posición del logo de forma consistente en todo el contenido visual de las páginas de empresa

Publicaciones de documento / carrusel

La función de carrusel de PDF/documento de LinkedIn permite presentaciones de varias páginas donde cada "diapositiva" es una imagen completa. Estas publicaciones generan muchas impresiones al desplazarse. Buenas prácticas:

  • Usa 1080×1080 px (cuadrado) para cada página, consistente y ocupa todo el feed
  • Coloca un gancho atractivo en la página 1 (que se muestra como portada)
  • Usa una plantilla visual consistente en todas las páginas
  • Exporta como PDF con imágenes incrustadas a 150 DPI mínimo
Conclusión clave: Para publicaciones en el feed de LinkedIn, usa imágenes cuadradas de 1080×1080 px para máxima visibilidad. Sube PNG para gráficos con texto, JPEG (q85) para fotos. Mantén la foto de perfil y el logo de empresa centrados dentro de un recorte circular. El banner de portada (1584×396 para personal, 1128×191 para empresa) suele descuidarse, invierte en un diseño profesional aquí.
