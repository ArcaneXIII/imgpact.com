How to Convert Video to GIF

  1. Upload an MP4, WebM, or MOV file (up to 200 MB).
  2. Set the start and end times to extract only the clip you need.
  3. Choose output width and frame rate (10 fps is a good balance of quality and file size).
  4. Click Convert to GIF. FFmpeg.wasm processes everything locally.
  5. Download the resulting animated GIF.

Getting the Best Results

  • Clip length: Keep clips under 10 seconds for manageable GIF sizes.
  • Frame rate: 10–15 fps looks smooth without inflating file size unnecessarily.
  • Width: 480 px is standard for web GIFs; go lower for smaller files.
  • Colors: 256 colors gives the best quality; lower values produce smaller files.

About Video to GIF Conversion

imgpact converts video to GIF entirely in your browser using FFmpeg.wasm, a full port of FFmpeg compiled to WebAssembly. Your video file never leaves your device. The 200 MB limit exists due to browser memory constraints, not server limitations.
