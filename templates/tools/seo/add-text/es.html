Cómo añadir texto a una imagen

  1. Sube tu imagen (PNG, JPG, WebP o BMP).
  2. Escribe tu texto en el campo de entrada.
  3. Elige el tamaño de fuente, el color y las opciones de estilo.
  4. Haz clic en el lienzo para posicionar el texto, o introduce coordenadas X/Y manualmente.
  5. Haz clic en Integrar texto en imagen para aplicar el texto.
  6. Descarga la imagen final en PNG, JPG o WebP.

Consejos para texto superpuesto

  • El contraste importa: Usa texto blanco sobre fondos oscuros y viceversa.
  • Varias líneas: Añade texto varias veces para diferentes posiciones, tamaños o colores.
  • Marca de agua: Añade tu nombre o URL con baja opacidad para una marca de agua discreta.
  • Memes y leyendas: Usa una fuente grande y en negrita arriba o abajo, estilo impact clásico.
