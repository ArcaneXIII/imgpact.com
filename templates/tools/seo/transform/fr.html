Comment faire pivoter ou retourner une image

  1. Importez votre image (PNG, JPG, WebP ou BMP).
  2. Cliquez sur un bouton de transformation, l'aperçu se met à jour instantanément.
  3. Enchaînez plusieurs transformations : deux rotations de 90° gauche équivalent à 180°.
  4. Sélectionnez votre format de sortie (garder l'original, PNG, JPG ou WebP).
  5. Cliquez sur Télécharger pour enregistrer le résultat.

Transformations disponibles

  • Rotation 90° gauche : Tourne l'image dans le sens antihoraire.
  • Rotation 90° droite : Tourne l'image dans le sens horaire, la correction la plus courante pour les photos prises de côté.
  • Rotation 180° : Retourne l'image à l'envers.
  • Retournement horizontal : Crée une image miroir de gauche à droite.
  • Retournement vertical : Crée un reflet de haut en bas.
