Comment analyser un GIF

  1. Importez votre GIF animé.
  2. imgpact analyse instantanément le fichier et affiche toutes ses métadonnées.
  3. Parcourez le tableau des images pour voir le délai, les dimensions et la méthode de rendu de chaque image.
  4. Utilisez les contrôles de lecture pour inspecter chaque image manuellement.

Comprendre les métadonnées GIF

  • Dimensions : Largeur et hauteur du canevas en pixels, toutes les images partagent le même canevas.
  • Nombre d'images : Nombre total d'images. Plus d'images = animation plus fluide et fichier plus grand.
  • Durée totale : Somme de tous les délais, la durée d'une boucle complète.
  • Nombre de boucles : 0 signifie boucle infinie ; tout autre nombre est le compte exact de répétitions.
  • Délai d'image : Temps en millisecondes d'affichage de chaque image. 100 ms = 10 ips.
  • Palette de couleurs : Le GIF utilise des couleurs indexées, jusqu'à 256 par image.
