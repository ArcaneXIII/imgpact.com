Comment appliquer des effets à une image

  1. Importez votre image (PNG, JPG, WebP ou BMP).
  2. Cliquez sur une carte d'effet pour la sélectionner, un aperçu apparaît immédiatement.
  3. Faites glisser le curseur Intensité pour régler la force de l'effet.
  4. Cliquez sur Appliquer l'effet pour valider l'effet sur l'image.
  5. Répétez avec d'autres effets pour les empiler, chaque application s'ajoute à la précédente.
  6. Choisissez votre format de sortie et cliquez sur Télécharger.

Effets disponibles expliqués

  • Niveaux de gris : Convertit l'image en noir et blanc.
  • Sépia : Ajoute une teinte brun chaud, effet photo vintage classique.
  • Flou : Adoucit les détails. Utile pour les arrière-plans ou les effets de profondeur de champ.
  • Netteté : Renforce les contours et les détails fins. Idéal pour les photos légèrement floues.
  • Luminosité : Rend l'image plus claire ou plus sombre dans l'ensemble.
  • Contraste : Augmente ou diminue la différence entre les zones claires et sombres.
  • Inverser : Inverse toutes les couleurs, crée un négatif de l'image.
  • Pixeliser : Crée un aspect mosaïque / pixel art.
