How to Create an Animated GIF

  1. Upload your images (PNG, JPG, or WebP).
  2. Reorder frames by dragging them in the timeline.
  3. Set the frame delay in milliseconds (100 ms = 10 fps; 50 ms = 20 fps).
  4. Choose infinite loop or enter a custom repeat count.
  5. Click Create GIF and download your animation.

Tips for Smaller, Sharper GIFs

  • Fewer colors: GIF supports up to 256 colors per frame, simple graphics compress best.
  • Crop first: Reduce the canvas to only what's needed before creating the GIF.
  • Longer frame delays: Slowing down the animation reduces file size without sacrificing quality.
  • Consistent dimensions: Using the same size for all frames prevents rendering artifacts.

About the GIF Maker

imgpact's GIF Maker runs entirely in your browser using a Rust + WebAssembly engine. No files are uploaded, everything is processed locally on your device, so there are no file size limits and your images remain 100% private.
